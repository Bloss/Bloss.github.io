# 场景

有如下对象:

const plugins = {
  hover: {
    state: {},
    reducer: () => {},
    component: () => <div>This is hover plugin</div>
  },
  drag: {
    state: {},
    reducer: () => {},
    component: () => <div>This is drag plugin</div>
  }
}

这是一个插件系统, 我们有如上两个插件, 第一个用来当我们 hover 在元素上的时候, 显示一些 hover 样式, 第二个用来当我们拖拽该元素的时候, 显示一些拖拽的工具栏.
